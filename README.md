# Spam SMS Detection using Machine Learning

##  Project Description
This project detects whether an SMS message is **Spam** or **Not Spam (Ham)** using Machine Learning.

##  Objective
To classify SMS messages accurately using Natural Language Processing (NLP) and the Naive Bayes algorithm.

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib

##  Dataset
SMS Spam Collection Dataset

## Machine Learning Algorithm
- TF-IDF Vectorizer
- Multinomial Naive Bayes

## Model Accuracy
**96.68%**

##  Project Files
- spam_sms_detection.py
- spam.csv
- model.pkl
- vectorizer.pkl
- requirements.txt
- README.md

##  How to Run

1. Install dependencies:
```
pip install -r requirements.txt
```

2. Run the project:
```
python spam_sms_detection.py
```
